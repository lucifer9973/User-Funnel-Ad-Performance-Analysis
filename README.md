# 📊 User Funnel & Ad Performance Analysis (Excel Project)

## 🚀 Overview

This project focuses on analyzing user behavior and ad performance using Excel. The goal was to understand how users move through different stages of a funnel and identify drop-offs that impact conversions.

## 🎯 Objective

- Analyze user journey from impressions → clicks → signups → engagement → payments
- Track key ad metrics like CTR, CPC, and conversion rates
- Identify bottlenecks and suggest improvements

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Charts & Visualization
- Conditional Formatting

## 📈 Key Metrics Analyzed

- Click Through Rate (CTR)
- Cost Per Click (CPC)
- Conversion Rates (Signup, Engagement, Payment)
- Funnel Drop-offs

## 🔍 Insights

- Significant drop-off observed between signup and engagement stage
- Ads with higher CTR did not always result in higher conversions
- Payment stage showed the highest drop-off, indicating friction in the final step

## 💡 Improvements Suggested

- Simplify onboarding to reduce signup friction
- Optimize payment flow to improve completion rate
- Focus on high-converting ad segments instead of just high CTR

## 📌 Outcome

This project demonstrates how data can be used to analyze user funnels, evaluate ad performance, and drive actionable growth decisions.
